**API Request Flow
1. Client Request → Load Balancer → Spring Boot API
2. API → Authentication/Authorization Check
3. API → Service Layer → Business Logic Validation
4. Service → Repository Layer → Database Query
5. Database → Repository → Service → API → Client Response

**Database Schema Design
employees (id, name, email, department_id, manager_id, created_at)
leave_types (id, name, max_days_per_year, carry_forward_allowed)
leave_requests (id, employee_id, leave_type_id, start_date, end_date, status, reason, submitted_at, reviewed_at, reviewer_id)
leave_balances (employee_id, leave_type_id, allocated_days, used_days, remaining_days, year)
departments (id, name, manager_id)
